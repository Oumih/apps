# ダッシュボード

## 概要
特定の1日の売上・キャストランキングを確認する日次ビュー。
前後の日に移動できる。

## アクセス権限
owner

## 表示項目
| 項目 | 内容 |
|------|------|
| 日付ナビゲーター | `< 2026/04/15 >` 形式。前後の日に移動できる |
| 日次売上合計 | 選択した日の sales.amount の合計 |
| キャストランキング | その日の売上が多い順にキャストを表示（sales を cast_id でグループ化・集計） |
| 各キャストの売上 | 名前・売上金額 |

## 操作・アクション
| 操作 | 結果 |
|------|------|
| `<` ボタン | 前日のデータを表示 |
| `>` ボタン | 翌日のデータを表示 |
| 日付をタップ | カレンダーピッカーで任意の日付に移動 |

## 備考
- デフォルトは今日の日付を表示
- データが存在しない日は「この日の売上データはありません」と表示
