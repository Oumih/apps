# 売上入力画面

## 概要
1回の入力で複数キャストの売上をまとめて登録できる画面。
登録完了後は同じ画面に留まり、続けて入力できる。

## アクセス権限
owner・admin

## 表示項目
| 項目 | 内容 |
|------|------|
| 日付 | 入力対象の日付（デフォルト: 今日） |
| 入力行 | キャスト選択 + 金額入力（複数行） |
| 行追加ボタン | 「+ 行を追加」で入力行を増やす |
| 行削除ボタン | 各行の右端に配置 |
| メモ | セッション全体へのメモ（任意） |
| 登録ボタン | 全行をまとめてsalesテーブルに登録 |

## 入力行の項目
| 項目 | 種別 | 必須 |
|------|------|------|
| キャスト | ドロップダウン（is_active = true のキャストのみ） | ✅ |
| 売上金額 | 数値入力（¥表示） | ✅ |

## バリデーション
| 条件 | エラーメッセージ |
|------|----------------|
| 行が0件 | 「1件以上入力してください」 |
| キャスト未選択 | 「キャストを選択してください」 |
| 金額未入力 | 「金額を入力してください」 |
| 金額が0以下 | 「0より大きい金額を入力してください」 |
| 同じキャストが重複 | 「同じキャストが重複しています」 |

## 登録後の動作
- 「登録しました」のトースト通知を表示
- 入力行をリセットして同じ画面に留まる（日付はそのまま）
