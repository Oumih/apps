# 店舗管理画面

## 概要
オーナーが自分の店舗情報を閲覧する画面。
編集はアプリ管理者（開発者）がSupabaseダッシュボードから行うため、この画面は閲覧のみ。

## アクセス権限
owner

## 表示項目
| 項目 | 内容 |
|------|------|
| 店舗名 | stores.name |
| 住所 | stores.address |
| 電話番号 | stores.phone |
| 営業時間 | stores.open_time 〜 stores.close_time |
| 登録日 | stores.created_at |

## 備考
- すべての項目は読み取り専用（編集ボタンなし）
- 情報の変更が必要な場合はアプリ管理者に連絡する旨を画面下部に表示
- 未入力の項目は「未設定」と表示する
