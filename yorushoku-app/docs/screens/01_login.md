# ログイン画面

## 概要
店舗単位で発行されたアカウントでログインする画面。

## アクセス権限
全員（未認証状態）

## 表示項目
| 項目 | 種別 | 必須 |
|------|------|------|
| アプリロゴ・名称 | 表示 | - |
| メールアドレス | テキスト入力 | ✅ |
| パスワード | パスワード入力 | ✅ |
| ログインボタン | ボタン | - |
| エラーメッセージ | 表示（条件付き） | - |

## バリデーション
| 条件 | エラーメッセージ |
|------|----------------|
| メール未入力 | 「メールアドレスを入力してください」 |
| パスワード未入力 | 「パスワードを入力してください」 |
| 認証失敗 | 「メールアドレスまたはパスワードが正しくありません」 |

## ログイン後の遷移
| role | 店舗数 | 遷移先 |
|------|--------|--------|
| owner | 複数 | 店舗選択画面 |
| owner | 1店舗 | ダッシュボード |
| admin | - | 売上入力画面 |
