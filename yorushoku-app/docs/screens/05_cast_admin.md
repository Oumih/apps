# キャスト・Admin管理画面

## 概要
キャストとadminアカウントをタブで切り替えて管理する画面。
追加・編集はモーダルで行う。

## アクセス権限
owner

---

## キャストタブ

### 表示項目
| 項目 | 内容 |
|------|------|
| キャスト一覧 | 在籍中（is_active = true）のキャストを表示 |
| 名前・読み仮名 | |
| 入店日 | |
| 累計売上 | sales を cast_id で集計 |
| 追加ボタン | モーダルを開く |
| 編集ボタン | モーダルを開く |
| 退店ボタン | is_active を false に更新（論理削除） |

### キャスト追加・編集モーダル
| 項目 | 種別 | 必須 |
|------|------|------|
| 名前（源氏名） | テキスト入力 | ✅ |
| 読み仮名 | テキスト入力 | ❌ |
| プロフィール | テキストエリア | ❌ |
| 入店日 | 日付入力 | ❌ |

### バリデーション
| 条件 | エラーメッセージ |
|------|----------------|
| 名前未入力 | 「名前を入力してください」 |

---

## Adminタブ

### 表示項目
| 項目 | 内容 |
|------|------|
| admin一覧 | profiles で role = 'admin' のユーザーを表示 |
| 名前 | |
| メールアドレス | |
| 追加ボタン | モーダルを開く |
| 削除ボタン | adminアカウントを削除 |

### Admin追加モーダル
| 項目 | 種別 | 必須 |
|------|------|------|
| 名前 | テキスト入力 | ✅ |
| メールアドレス | テキスト入力 | ✅ |
| パスワード | パスワード入力 | ✅ |

### バリデーション
| 条件 | エラーメッセージ |
|------|----------------|
| 名前未入力 | 「名前を入力してください」 |
| メール未入力・形式不正 | 「正しいメールアドレスを入力してください」 |
| パスワード8文字未満 | 「パスワードは8文字以上で入力してください」 |
| メール重複 | 「このメールアドレスはすでに使用されています」 |
