# 売上管理画面

## 概要
月別・年別の売上推移を確認する集計ビュー。
過去データの分析に使う。

## アクセス権限
owner

## 表示項目
| 項目 | 内容 |
|------|------|
| 表示切り替えタブ | 月別 / 年別 |
| 売上グラフ | 選択した期間の売上推移グラフ（棒グラフ） |
| 売上合計 | 表示期間の合計金額 |
| 売上平均 | 表示期間の平均金額 |
| 売上最高 | 表示期間の最高金額 |
| 売上テーブル | 期間ごとの売上一覧（期間・金額） |
| キャスト別売上 | 表示期間内のキャストごとの売上合計・ランキング |

## 操作・アクション
| 操作 | 結果 |
|------|------|
| 月別タブ | 過去12ヶ月の月別売上グラフを表示 |
| 年別タブ | 年ごとの売上グラフを表示 |

## データの集計方法
- 売上合計: `SELECT SUM(amount) FROM sales WHERE store_id = ? AND date BETWEEN ? AND ?`
- キャスト別: `SELECT cast_id, SUM(amount) FROM sales GROUP BY cast_id ORDER BY SUM(amount) DESC`
